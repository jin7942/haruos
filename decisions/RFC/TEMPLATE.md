# RFC-NNN: 제목

| 버전 | 변경내용 | 작성자 | 수정일 |
| --- | --- | --- | --- |
| v1.0 | 초기 작성 | 김진범 | YYYY-MM-DD |

- **상태**: 제안 / 논의 중 / 합의 / 확정으로 승격 / 폐기 / 보류
- **일자**: YYYY-MM-DD
- **작성자**: 김진범
- **관련 ADR**: (없음 — 합의 후 ADR 승격 예정 / 또는 ADR-NNN 링크)

## 요약

무엇을 제안하는가 (2~3문장)

## 배경

현재 구조 / 식별된 문제 / 왜 지금 논의가 필요한가

## 검토 선택지

- 옵션 1 — ... (장점/단점)
- 옵션 2 — ... (장점/단점)  ※ 기각안도 이유와 함께
- 옵션 3 — ...

## 추천

어느 옵션을 왜 미는가

## 미해결 질문

→ 항목이 무거우면 후속 RFC로 분리 (RFC-002, RFC-003 …)
