# RFC 목록

논의가 필요한 의사결정(대안이 다투거나 합의가 필요하거나 되돌리기 비싼 결정)을
RFC로 기록한다. `합의` 도달 시 ADR로 승격한다 (→ `decisions/README.md` §승격 규칙).

`TEMPLATE.md`를 복사해 `RFC-NNN-제목.md`로 작성한다.

## 상태 정의

| 상태 | 의미 |
| --- | --- |
| 제안 | 처음 작성, 논의 시작 전 |
| 논의 중 | 팀에서 검토·토론 진행 중 |
| 합의 | 방향 합의됨, ADR 승격 대기 |
| 확정으로 승격 | ADR로 옮겨짐 (이 RFC는 히스토리로 보존, 상단에 ADR 링크 명시) |
| 폐기 | 채택하지 않기로 결정 |
| 보류 | 결정 보류, 추후 재검토 |
