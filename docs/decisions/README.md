# Decisions

프로젝트의 의사결정 관련 문서를 모은 곳.

## 폴더 구성

| 폴더 | 용도 |
| --- | --- |
| [`ADR/`](ADR/) | **확정된** 의사결정 기록 (Architecture Decision Record) |
| [`RFC/`](RFC/) | **논의 중**인 제안서 (Request For Comments) |

## 워크플로우

```
RFC (제안/논의) ── 팀 합의 ──> ADR (확정 기록) ──> 코드/문서 반영
       │
       └── 합의 안 됨 ──> 폐기 / 보류
```

- 새 아이디어·제안은 **RFC**로 시작
- 팀 논의 후 합의되면 새 **ADR**로 확정 기록 작성 (RFC는 히스토리 보존)
- 합의 안 되면 RFC 상태를 `폐기` 또는 `보류`로 변경

## 작성 규칙

각 폴더의 `README.md`(목록 인덱스)와 `TEMPLATE.md`(작성 템플릿) 참조.

상세 작성 규칙은 프로젝트 루트 `CLAUDE.md`의 ADR/RFC 섹션에 명시되어 있다.
