## 작성 방법

[`TEMPLATE.md`](TEMPLATE.md)를 복사해서 `ADR-NNN-제목.md` 형식으로 작성한다.

- 번호: 순차 부여 (001, 002, ...)
- 새 ADR 작성 시 이 README 표에 한 줄 추가
- 확정된 ADR은 수정하지 않는다. 변경 시 새 ADR을 작성하고 기존을 `대체됨` 상태로 변경 (후속 ADR 링크 명시)

## 상태 정의

| 상태 | 의미 |
| --- | --- |
| **제안** | 작성 직후, 합의 전 (일반적으로는 RFC를 거치는 것을 권장) |
| **확정** | 합의 완료, 적용 중 |
| **대체됨** | 후속 ADR로 대체. 후속 ADR 링크를 본문에 명시 |
| **폐기** | 더 이상 유효하지 않음 |

논의 중인 제안은 [`../RFC/`](../RFC/) 폴더 참조.
