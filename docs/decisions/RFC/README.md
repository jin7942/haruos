## 작성 방법

[`TEMPLATE.md`](TEMPLATE.md)를 복사해서 `RFC-NNN-제목.md` 형식으로 작성한다.

- 번호: 순차 부여 (001, 002, ...)
- 제목: 짧게, 한국어 권장
- 새 RFC 작성 시 이 README 표에 한 줄 추가

## 상태 정의

| 상태 | 의미 |
| --- | --- |
| **제안** | 처음 작성, 논의 시작 전 |
| **논의 중** | 팀에서 검토·토론 진행 중 |
| **합의** | 방향 합의됨, ADR 승격 대기 |
| **확정으로 승격** | ADR로 옮겨짐 (이 RFC는 히스토리로 보존) |
| **폐기** | 채택하지 않기로 결정 |
| **보류** | 결정 보류, 추후 재검토 |

확정으로 승격된 경우, 대응하는 ADR 링크를 RFC 문서 상단에 명시한다.
